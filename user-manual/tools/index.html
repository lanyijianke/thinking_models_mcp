天机思维模型工具体系

天机MCP服务器包含丰富的工具集，提供思维模型的浏览、查询、推荐和应用，帮助用户系统地解决复杂问题。

探索类工具

查询与浏览工具
list-models search-models get-model-info get-related-models
这组工具可以帮助用户浏览模型库、查找相关模型，并获取详细的模型信息
推荐工具
recommend-models-for-problem
基于问题关键词推荐适合解决特定问题的思维模型

问题解决类工具

推理工具
explain-reasoning-process interactive-reasoning generate-validate-hypotheses
这组工具提供结构化的思考过程，帮助用户进行更深入、更系统的分析和推理

创建类工具

模型创建工具
create-thinking-model emergent-model-design update-thinking-model
这组工具允许高级用户创建自己的思维模型，或通过组合现有模型的关键概念创建新模型

学习与反馈工具

学习系统工具
record-user-feedback detect-knowledge-gap get-model-usage-stats analyze-learning-system
这组工具可以记录用户反馈，分析知识缺口，并提供模型使用统计数据，帮助系统不断优化和学习

系统工具

系统管理工具
get-server-version count-models get-started-guide
这组工具提供系统级信息，如服务器版本、模型数量统计，以及新手入门指南

工具使用指南

天机思维模型工具系统以MCP协议为基础，为不同使用场景提供了一系列专门工具。以下是针对不同需求的工具使用建议：

模型探索场景

当您初次使用天机或希望了解可用的思维模型时，可以通过list-models工具获取模型列表，使用search-models根据关键词查找特定模型， 并通过get-model-info深入了解某个模型的详情。

问题解决场景

面对特定问题时，可先使用recommend-models-for-problem获取适合该问题的模型推荐， 然后应用explain-reasoning-process或interactive-reasoning工具进行系统性思考和分析。

高级使用场景

对于熟悉思维模型体系的高级用户，可以尝试通过create-thinking-model或emergent-model-design工具创建自定义思维模型， 并使用record-user-feedback提供反馈，帮助系统不断改进。

MCP工具调用提示

所有工具都可以通过MCP协议调用，建议将工具作为AI助手的增强能力使用，而非直接与工具交互。例如，当需要解决复杂问题时， 可以告诉AI助手："请使用天机的思维模型工具，为我分析这个商业策略问题"，AI助手会自动调用相关工具，提供系统化的思考过程。
