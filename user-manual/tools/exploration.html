探索类工具

探索类工具是"天机"的发现引擎，帮助用户探索、了解并建立思维模型知识体系。这些工具专注于帮助用户找到合适的思维模型，并深入了解其内容和应用方法。

模型列表浏览 (list-models)

探索思维工具库

功能描述

list-models 工具提供了对"天机"思维模型库的全面浏览功能，允许用户按类别、语言等条件查看可用的思维模型列表。这是用户初始探索思维模型库的重要入口，特别适合希望系统性了解现有思维工具的用户。

参数说明

参数名 类型 必填 默认值 说明
category 字符串 否 - 主分类名称，用于过滤特定类别的模型
subcategory 字符串 否 - 子分类名称（需同时提供主分类）
lang 字符串 否 "zh" 语言代码，支持 "zh"(中文) 或 "en"(英文)
limit 数字 否 100 返回结果数量限制

使用示例

// 列出所有中文思维模型
{
  "lang": "zh"
}

// 列出特定类别的思维模型
{
  "category": "决策思维",
  "lang": "zh"
}

// 列出特定子类别的思维模型
{
  "category": "系统思维",
  "subcategory": "反馈循环",
  "lang": "zh",
  "limit": 10
}

实际应用场景

  • 初学者系统性了解可用思维工具
  • 研究特定领域（如决策思维）的相关模型
  • 教学环境中创建思维模型课程大纲
  • 构建个人思维工具体系的第一步
提示： 对初次使用天机的用户，建议先使用此工具浏览全部模型分类，建立对思维模型体系的整体认知，然后再深入特定类别。

关键词模型搜索 (search-models)

精确查找思维工具

功能描述

search-models 工具允许用户通过关键词在思维模型库中进行文本搜索，快速定位相关模型。相比于分类浏览，关键词搜索更加直接且高效，特别适合已经对思维模型有一定了解、寻找特定模型的用户。

参数说明

参数名 类型 必填 默认值 说明
query 字符串 是 - 搜索关键词
lang 字符串 否 "zh" 语言代码，支持 "zh"(中文) 或 "en"(英文)
limit 数字 否 10 返回结果数量限制

使用示例

// 搜索与"偏见"相关的思维模型
{
  "query": "偏见",
  "lang": "zh"
}

// 搜索与"系统"相关的思维模型，限制结果为5个
{
  "query": "系统",
  "lang": "zh",
  "limit": 5
}

实际应用场景

  • 快速查找特定主题相关的思维模型
  • 解决问题时寻找相关的思考工具
  • 研究特定概念在不同思维模型中的应用
  • 准备演讲或写作时搜集相关思维框架
提示： 尝试使用多种相关关键词进行搜索，以获得更全面的结果。搜索不仅匹配模型名称，还会搜索模型描述、用途和标签等字段。

获取模型详情 (get-model-info)

深入了解模型内容

功能描述

get-model-info 工具提供思维模型的详细信息，包括定义、用途、示例、可视化内容等。用户可以指定需要返回的内容字段，从基本信息到完整细节都可定制。这是深入学习和应用特定模型的关键工具。

参数说明

参数名 类型 必填 默认值 说明
model_id 字符串 是 - 思维模型的唯一ID
fields 字符串数组 否 ["basic"] 需要返回的字段，如 'all'(全部), 'basic'(基本信息), 'detail'(详细信息), 'teaching'(教学内容), 'warnings'(注意事项), 'visualizations'(可视化)
lang 字符串 否 "zh" 语言代码，支持 "zh"(中文) 或 "en"(英文)

使用示例

// 获取模型的基本信息
{
  "model_id": "first_principles_thinking",
  "lang": "zh"
}

// 获取模型的所有信息
{
  "model_id": "first_principles_thinking",
  "fields": ["all"],
  "lang": "zh"
}

// 只获取模型的可视化和教学内容
{
  "model_id": "first_principles_thinking",
  "fields": ["visualizations", "teaching"],
  "lang": "zh"
}

实际应用场景

  • 深入学习特定思维模型的原理和应用方法
  • 准备培训材料时获取模型的教学内容和可视化
  • 将模型应用于实际问题前了解其局限性和常见陷阱
  • 研究不同语言版本的思维模型表达差异
提示： 使用 ["all"] 获取模型的完整信息可能返回较大的数据量。如果只需要特定信息，建议指定所需字段以提高效率。

相关模型发现 (get-related-models)

扩展思考维度

功能描述

get-related-models 工具基于语义相似性和模型关联性，推荐与指定模型相关的其他思维模型。此工具帮助用户探索思维模型之间的联系，构建思维模型网络，发现更多相关的思考方法。

参数说明

参数名 类型 必填 默认值 说明
model_id 字符串 是 - 源思维模型的唯一ID
limit 数字 否 5 返回结果数量限制
use_enhanced_similarity 布尔值 否 true 是否使用增强的相似度评估（如果为false则使用基础算法）
lang 字符串 否 "zh" 语言代码，支持 "zh"(中文) 或 "en"(英文)

使用示例

// 获取与"第一性原理思维"相关的模型
{
  "model_id": "first_principles_thinking",
  "lang": "zh"
}

// 获取更多相关模型并使用基础相似度算法
{
  "model_id": "first_principles_thinking",
  "limit": 10,
  "use_enhanced_similarity": false,
  "lang": "zh"
}

实际应用场景

  • 扩展单一思维模型的应用场景和视角
  • 构建特定领域的思维模型知识网络
  • 发现互补的思考方法，组合使用解决复杂问题
  • 创建思维模型学习路径，从基础到进阶模型
提示： 增强的相似度评估算法会考虑模型之间更深层的语义关联，而不只是表面的文本相似性，通常能提供更有价值的推荐结果。

获取模型分类 (get-categories)

建立思维体系认知

功能描述

get-categories 工具返回所有思维模型的分类信息，包括主分类和子分类，帮助用户了解思维模型的整体分类体系。这是构建系统化思维工具认知的重要工具。

参数说明

参数名 类型 必填 默认值 说明
lang 字符串 否 "zh" 语言代码，支持 "zh"(中文) 或 "en"(英文)

使用示例

// 获取中文思维模型分类
{
  "lang": "zh"
}

// 获取英文思维模型分类
{
  "lang": "en"
}

实际应用场景

  • 了解思维模型的整体分类体系
  • 规划系统性学习思维模型的路径
  • 为团队或组织创建思维工具分类参考
  • 设计思维模型培训课程的模块划分
提示： 配合 list-models 工具使用可以先了解整体分类，然后有针对性地浏览各个类别的具体模型。

探索技巧

有效地使用探索类工具可以显著提升您发现和学习思维模型的效率。以下是一些建议的探索流程：

  1. 首先使用 get-categories 了解思维模型的整体分类体系
  2. 使用 list-models 浏览感兴趣类别的具体模型
  3. 通过 get-model-info 深入学习特定模型的详细内容
  4. 利用 get-related-models 发现相关模型，扩展思考维度
  5. 随时使用 search-models 快速查找特定关键词相关的模型

通过这种系统化的探索方法，您可以逐步建立个人的思维模型知识体系，并在实际应用中更灵活地选择合适的思考工具。
