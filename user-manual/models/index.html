如何使用思维模型

可以通过以下方式选择合适的思维模型：

  • 使用 bb7_search-models 工具搜索相关关键词
  • 使用 bb7_recommend-models-for-problem 工具获取针对特定问题的推荐模型
  • 通过 bb7_list-models 工具按类别浏览模型
  • 使用 bb7_detect-knowledge-gap 工具分析问题并推荐合适模型

选择模型后，使用 bb7_get-model-info 工具获取模型的详细信息：

{
  "model_id": "first_principles_thinking",
  "fields": ["all"]
}

这将返回模型的定义、目的、应用场景、使用方法、可视化内容等详细信息。

将思维模型应用于问题解决的几种方法：

  • 在提问中明确指定使用特定模型：
    "请使用SWOT分析模型帮我分析这个商业计划"
  • 使用 bb7_explain-reasoning-process 工具获取模型应用的详细推理过程
  • 组合多个模型：
    "请结合波特五力模型和PESTEL分析帮我评估这个市场"

使用 bb7_record-user-feedback 工具提供关于模型使用体验的反馈，这将帮助系统不断优化模型推荐和应用效果。

例如：

{
  "modelIds": ["five_whys"],
  "context": "产品开发问题分析",
  "feedbackType": "helpful",
  "comment": "五个为什么分析法帮助我找到了问题的根本原因"
}
