搜索API使用示例

按关键词搜索模型

使用 bb7_search-models 工具在模型库中搜索关键词：

{
  "query": "系统思维",
  "lang": "zh",
  "limit": 10
}

示例响应:

{
  "models": [
    {
      "id": "systems_thinking",
      "name": "系统思维",
      "category": "系统与战略思维",
      "relevance": 0.98,
      "definition": "一种用于分析和解决复杂问题的方法，通过识别系统中的各组成部分及其相互关系，建立全面理解。"

}
按问题推荐模型

使用 bb7_recommend-models-for-problem 工具针对特定问题推荐模型：

{
  "problem_keywords": ["市场分析", "竞争对手", "战略规划"],
  "problem_context": "我需要分析我们公司在新市场的竞争环境，并制定合适的市场进入战略",
  "lang": "zh",
  "limit": 5
}

示例响应:

    {
      "id": "porters_five_forces",
