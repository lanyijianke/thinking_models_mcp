Cherry Studio 环境配置

本文档详细说明如何在 Cherry Studio 集成开发环境中配置 MCP 服务器

配置流程

请确保您使用的是支持 MCP 的 Cherry Studio 版本。
  1. 访问服务器配置界面：
    • 点击"添加服务器"按钮
  2. 导入服务器配置：
    • 在配置方式中选择"JSON导入"选项
    • 将预先准备的配置代码粘贴到指定区域
  3. 完成配置：
    • 保存配置信息
    • 返回主界面，新添加的 MCP 服务器将在服务器列表中显示
    • 可以通过对话界面直接调用配置的服务器
