Claude 桌面应用配置指南

本指南将帮助您在 Claude 桌面应用中配置和使用"天机"思维模型 MCP 服务器，让您在日常对话中轻松访问强大的思维框架和工具。Claude 桌面应用支持 MCP 协议，可与"天机"思维模型服务器无缝集成。

Claude MCP 思维模型 Anthropic

配置步骤概览

1

安装 Claude 桌面应用

如果您尚未安装 Claude 桌面应用，请先下载并安装：

下载 Claude 桌面应用

Claude 桌面应用是 Anthropic 提供的官方客户端，支持 Model Context Protocol (MCP)，可以通过插件和扩展功能增强 Claude 的能力。

注意：请确保您安装的是支持 MCP 的最新版本 Claude 桌面应用。

2

启动"天机"MCP 服务器

确保"天机"思维模型 MCP 服务器已正确安装并且正在运行：

# 如果使用全局安装
thinking-models-server

# 如果使用本地安装
npm run start-thinking-models

建议使用网络模式启动服务器，因为 Claude 桌面应用通常更好地支持网络连接：

# 使用指定端口启动（Windows）
set THINKING_MODELS_PORT=3456
thinking-models-server

# 使用指定端口启动（Linux/macOS）
THINKING_MODELS_PORT=3456 thinking-models-server

服务器启动后，应该会显示一条消息表明服务器已经在监听连接。

3

在 Claude 中配置 MCP 连接

打开 Claude 桌面应用，配置 MCP 服务器连接：

  1. 点击左下角的设置图标（⚙️）打开设置面板
  2. 找到"扩展"或"MCP 设置"部分（具体名称可能因版本而异）
  3. 点击"添加 MCP 服务器"或类似按钮
  4. 填写以下信息：
配置项 值 说明
名称 天机思维模型服务器 服务器的显示名称
地址 localhost:3456 如果使用网络模式，填写服务器地址和端口
命令（可选） thinking-models-server 如果不使用网络模式，可提供命令行
Claude 桌面应用的 MCP 配置界面可能因版本而有所不同。如果找不到相关设置，请检查应用是否为最新版本，或参考 Claude 官方文档查找当前版本的 MCP 配置方法。

4

连接到"天机"服务器

完成配置后，在 Claude 桌面应用中连接到"天机"服务器：

  1. 在设置面板中找到您添加的"天机思维模型服务器"
  2. 点击"连接"或开关按钮启用连接
  3. 如果连接成功，状态指示器将变为绿色或显示"已连接"

连接成功指示

连接成功后，Claude 应用通常会显示已连接到"天机思维模型服务器"的通知，以及服务器提供的功能列表或工具清单。

5

使用"天机"思维模型

成功连接后，可以在 Claude 中使用"天机"思维模型工具：

直接在对话中使用

在 Claude 的对话框中，您可以直接询问与"天机"思维模型相关的问题：

示例问题：

  • "天机"中有哪些关于系统思维的模型？
  • 使用SWOT分析思维模型帮我分析一个创业项目
  • 我需要一个框架来理解复杂系统的反馈循环，有什么推荐？
  • 请基于"天机"的思维模型库，推荐适合解决资源分配问题的3-5个模型

使用工具面板（如果支持）

较新版本的 Claude 桌面应用可能提供工具面板或快捷方式：

  1. 查找对话界面中的工具栏或工具按钮
  2. 查找"天机"或"思维模型"相关工具
  3. 使用工具面板直接访问特定的思维模型或功能
提示： 由于"天机"提供了多种思维工具，建议先查阅工具文档，了解各类思维工具的功能和使用方式，这样在 Claude 中可以更有针对性地调用合适的工具。

使用技巧与最佳实践

清晰指定工具

在 Claude 中使用"天机"时，明确指定您需要的工具可以获得更准确的结果：

请使用"天机"的 get-categories 工具查看所有思维模型分类，然后使用 list-models 工具列出决策类下的所有模型

设置持续会话

在 Claude 中，可以设置一个专门使用"天机"工具的会话：

  1. 创建一个新对话
  2. 首先告诉 Claude：在这个会话中，我希望你优先使用"天机"思维模型服务器提供的工具来回答我的问题。
  3. 保存这个对话以便日后继续使用

结合 Claude 的强大理解能力

Claude 拥有强大的语言理解能力，可以与"天机"的思维模型产生良好的协同效应：

请使用"天机"中的系统思维模型分析这个问题，然后用你自己的理解对结果进行深度解读，特别是结合最新的研究见解和实际应用案例。

工作流模式

创建一个多步骤的思维工作流：

我想分析一个复杂决策问题。请按以下步骤帮我：
1. 使用"天机"的 recommend-models-for-problem 工具为我推荐适合的模型
2. 然后使用 get-model-info 获取推荐模型的详细信息
3. 最后引导我使用这些模型构建一个分析框架

常见问题与排查

可能的原因和解决方案：

  • 版本过旧：确保使用最新版本的 Claude 桌面应用，旧版本可能不支持 MCP
  • 功能未启用：某些地区或账户可能需要额外启用 MCP 功能
  • 设置位置：尝试查找"扩展"、"插件"或"高级设置"等菜单项

访问 Claude 官方文档 查看最新的 MCP 配置说明。

可能的原因和解决方案：

  • 连接问题：尽管显示已连接，但可能实际连接不稳定
  • 命令格式：尝试更明确的命令格式，如"使用天机的get-categories工具"
  • 服务器状态：检查服务器是否正常响应，尝试重启服务器

断开并重新连接服务器，或者创建新的对话可能会解决临时问题。

可能的原因和解决方案：

  • 首次授权：首次连接到新的 MCP 服务器时，Claude 可能会请求授权确认
  • 权限设置：检查 Claude 设置中的服务器权限设置是否正确
  • 安全策略：某些企业环境可能限制外部连接，检查网络设置

如果系统请求授权，请仔细阅读权限内容，确认无误后授权"天机"服务器。

Claude 与思维模型的优势组合

Claude 作为一款强大的对话式 AI 助手，与"天机"思维模型的结合带来独特优势：

  • Claude 的自然语言理解能力可以准确捕捉问题核心，"天机"提供结构化思维框架
  • 复杂决策问题可以通过对话形式层层深入分析，并应用适当的思维模型
  • Claude 可以结合上下文，将"天机"的思维模型适应到您的具体场景
  • 日常对话中可以随时调用专业思维工具，提升思考质量
