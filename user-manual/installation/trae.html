Trae 开发环境配置

本文档详细说明如何在 Trae 开发环境中配置 MCP 服务器连接

配置流程

配置前请确保您的 Trae 环境已正确安装并具有适当的访问权限。
  1. 访问 Trae 配置界面：
    • 展开左侧导航栏
    • 找到系统设置图标（齿轮图标）
    • 选择 MCP 服务器配置选项
  2. 在服务器管理界面中：
    • 点击"添加服务器"按钮
    • 选择"手动配置"模式
  3. 在配置编辑器中：
    • 按照指定格式输入或粘贴服务器配置信息
    • 验证配置格式的正确性
  4. 完成配置后，系统会自动将新服务器添加到 MCP 服务器管理列表
